## Description
**WorldAuth** - Spigot plugin which allows you to add authorization on single server.

## Commands
* /_worldauth_ - plugin command \***NEW**\*
* /_changepassword_ (_/changepass_)
* /_logout_

## Permissions
* worldauth.reload

## Translation
